## Welcome to DirectoryEasy

DirectoryEasy is your comprehensive solution for managing directories and file structures efficiently.

## Quick setup

Follow these simple steps to get DirectoryEasy up and running:

### Step 1: Installation

<AccordionGroup>
  <Accordion icon="download" title="Install DirectoryEasy">
    Install DirectoryEasy using your preferred package manager:
    
    ```bash
    npm install directoryeasy
    # or
    yarn add directoryeasy
    # or
    pnpm add directoryeasy
    ```
  </Accordion>
  <Accordion icon="gear" title="Basic configuration">
    Set up your basic configuration:
    
    1. Create a configuration file
    2. Define your directory structure
    3. Set up your preferences
    
    <Tip>Configuration is optional - DirectoryEasy works out of the box with sensible defaults!</Tip>
  </Accordion>
</AccordionGroup>

### Step 2: First steps

<Accordion icon="play" title="Start using DirectoryEasy">
  1. Import DirectoryEasy in your project
  2. Initialize with your directory path
  3. Begin managing your directories!
</Accordion>

## What's next?

You're all set! DirectoryEasy is now ready to help you manage your directories efficiently.
